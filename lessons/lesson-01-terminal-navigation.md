# Lesson 1 - Terminal Navigation

## Commands Learned

### whoami

Displays the current logged-in user.

Example:

whoami

Output:

sam

---

### pwd

Displays current directory.

Example:

pwd

Output:

/home/sam

---

### ls

Lists files and folders.

Examples:

ls

ls -a

ls -l

---

### cd

Navigate between directories.

Examples:

cd ~

cd ..

cd Projects

---

### mkdir

Create directories.

Example:

mkdir LinuxPractice

---

### tree

Displays folder structure.

Example:

tree

## Real .NET Usage

* Navigate to ASP.NET Core projects
* Run dotnet commands
* Access Docker projects
* Access deployment folders
* View application logs

## Common Mistakes

### Wrong

cd~

### Correct

cd ~

(space required)

## Key Takeaway

Linux navigation is the foundation for working with .NET applications, Docker containers, Azure VMs, and CI/CD pipelines.
